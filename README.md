# words

### Tech used:
- Next
- MUI
- ReactSpeechKit
- React color
- Docker

### Installation
#### Docker
1. `docker pull stranavadavid/legendajezek`
2. `docker run -d -p 3000:3000 stranavadavid/legendajezek`

#### Manual installation
1. Clone this repo
2. Copy .env.example -> .env.local
3. Run `npm install`
4. Build the app `npm run build`
5. Start `npm start`
